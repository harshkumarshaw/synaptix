# Agent Learnings

When an agent is corrected by a human or discovers something useful, it's logged here so other agents can learn.

## Format

```markdown
### LEARN-NNN: [Brief title]

**Date:** YYYY-MM-DD
**Agent:** {agent-id}
**Type:** Correction / Discovery / Pattern

**Situation:**
What was the agent trying to do?

**What went wrong (or right):**
What the agent did or attempted.

**Correction / Insight:**
What should be done instead, or what was learned.

**Implications for other agents:**
What should other agents know?
```
